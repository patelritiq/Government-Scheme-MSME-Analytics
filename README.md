# Government Scheme & MSME Analytics Dashboard (MPSEDC – Personal Project)

## Overview
This project presents a data analytics dashboard designed to analyze the performance of selected government welfare schemes and the impact of MSME initiatives in Madhya Pradesh. The dashboard focuses on beneficiary reach, fund utilization efficiency, district-level performance, and MSME-driven employment generation.

The project was developed using **Power BI** and **Python** as part of a personal learning and analytics initiative.

---

## Key Objectives
- Analyze trends in welfare scheme beneficiaries and fund utilization
- Compare district-level performance across Madhya Pradesh
- Evaluate MSME investment and employment generation trends
- Demonstrate end-to-end analytics workflow from raw data to dashboard

---

## Dashboard Pages
1. **Executive Overview**
   - State-level KPIs
   - Year-wise beneficiary and fund utilization trends

2. **District Performance**
   - District-wise beneficiary distribution
   - Fund utilization efficiency comparisons

3. **MSME & Employment Impact**
   - MSME units, investment, and employment trends
   - Investment vs employment relationship analysis

---

## Data Sources
- Open Government Data Platform (data.gov.in)
- DBT scheme-wise beneficiary datasets
- MSME project approval and employment datasets
- Madhya Pradesh administrative hierarchy data

> Note: Due to limited public availability of recent scheme-wise data, synthetic data was generated using transparent and documented assumptions for analytical continuity.
